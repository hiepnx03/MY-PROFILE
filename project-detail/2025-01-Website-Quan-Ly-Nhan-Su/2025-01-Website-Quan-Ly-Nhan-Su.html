Website Quản Lý Nhân Sự

JDK 21

Giới thiệu dự án

Website Quản Lý Nhân Sự là hệ thống hỗ trợ quản trị nhân sự toàn diện, giúp doanh nghiệp quản lý hiệu quả các danh mục nhân sự, tài khoản, hợp đồng và lương. Dự án ứng dụng công nghệ hiện đại với khả năng xác thực, phân quyền chặt chẽ và các tính năng hỗ trợ báo cáo, xuất dữ liệu.

Chức năng chính:

        Quản lý nhân viên: thêm, sửa, xóa, phân loại theo phòng ban, chức vụ.
        Quản lý tài khoản: tạo tài khoản, phân quyền, đổi mật khẩu, khóa/mở tài khoản.
        Quản lý hợp đồng và lương.
        Xác thực và phân quyền bằng Spring Security + JWT.
        Chức năng "Quên mật khẩu" qua email, khóa tài khoản sau nhiều lần đăng nhập sai.
        Xuất dữ liệu báo cáo ra Excel.
        Tài liệu API bằng Swagger.

Công nghệ sử dụng:

        Backend: Spring Boot, Spring Security, JWT, Spring Data JPA
        Database: MySQL
        Frontend: React TypeScript
        API Docs: Swagger
