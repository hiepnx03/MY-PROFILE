Website Automation Tool Phát triển hệ thống tự động hoá tạo nội dung cho mạng xã hội, thu thập dữ liệu video từ YouTube, xử lý nội dung bằng AI và xuất kết quả sang Google Sheets. Hệ thống hỗ trợ tải lên và lưu trữ phương tiện qua FTP, đảm bảo chất lượng ảnh gốc.

Chức năng chính:

        Tóm tắt nội dung youtube
        Chatbot
        OCR
        backup media
        tải video tiktok, youtube

Công nghệ sử dụng:

        Backend: Spring Boot, Spring Security, JWT, Spring Data JPA
        Frontend: React TypeScript
        API Docs: Swagger
        Các công cụ tích hợp: LM Studio, n8n, FTP, YouTube API, Google Sheets API
